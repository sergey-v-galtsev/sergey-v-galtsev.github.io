Удаляет отображение заданного блока виртуальных страниц block. Физические фреймы, на которые не осталось других ссылок, освобождаются. Read more

source

unsafe fn remap(
    &mut self,
    old_block: Block<Page>,
    new_block: Block<Page>
) -> Result<()>

Перемещает отображение физических фреймов из old_block в new_block. Если изначально new_block содержал отображённые страницы, их отображение удаляется. Физические фреймы, на которые не осталось других ссылок, освобождаются. После работы BigAllocator::remap() виртуальные адреса old_block становятся недоступны. А содержимое памяти, которое ранее было доступно через old_block, становится доступным через new_block. Флаги доступа при этом не меняются. Read more
