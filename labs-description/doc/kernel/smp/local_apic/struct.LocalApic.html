Expand description

Memory–mapped I/O (MMIO) для работы с local APIC.

https://www.intel.com/content/dam/www/public/us/en/documents/manuals/64-ia-32-architectures-software-developer-vol-3a-part-1-manual.pdf, Chapter 10. https://wiki.osdev.org/APIC#Local_APIC_registers

Fields

id: Register

Позволяет узнать идентификатор local APIC и текущего CPU.

eoi: Register

Посылает local APIC сигнал о завершении работы обработчика прерывания — end of interrupt (EOI).

spurious: Register

Обработчик “фантомных прерываний” (Spurious interrupts).

https://wiki.osdev.org/APIC#Spurious_Interrupt_Vector_Register

error_status_register: Register

Регистр ошибок.

interrupt_command_lo: Register

Позволяет посылать другим процессорам прерывание — inter-processor interrupt (IPI). Эта (нижняя) часть двойного регистра задаёт флаги и дополнительные данные для IPI, см. InterruptCommand.

interrupt_command_hi: Register

Позволяет посылать другим процессорам прерывание — inter-processor interrupt (IPI). Эта (верхняя) часть двойного регистра задаёт целевой процессор для IPI.

lvt_timer: Register

Регистр таймера.

lvt_termal_sensor: Register

Регистр сенсора температуры.

lvt_performance_counter_overflow: Register

Регистр переполнения hardware performance counter (HPC).

timer_initial_count: Register

Начальное значение счётчика таймера.

timer_current_count: Register

Текущее значение счётчика таймера.

timer_divide_configuration: Register

Делитель счётчика таймера.
