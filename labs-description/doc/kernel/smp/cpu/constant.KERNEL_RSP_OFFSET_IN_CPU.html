Смещение внутри Cpu, по которому нужно сохранять rsp ядра, чтобы процессор переключал стек на него при возникновении прерываний в ненулевом кольце защиты. Ядро сохраняет там свой rsp, когда переключается в режим пользователя и восстанавливает свой rsp оттуда, когда возвращается из режима пользователя или получает он него системный вызов.
