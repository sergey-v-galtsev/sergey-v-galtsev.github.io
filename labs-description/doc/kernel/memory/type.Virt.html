Виртуальный адрес архитектуры x86-64.

Examples

Преобразования между Virt и базовыми типами

Преобразования между Virt и usize/u64.

let virt = Virt::new(0x123ABC)?;
assert!(virt == Virt::new_u64(0x123ABC)?);
assert!(virt.into_u64() == 0x123ABCu64);
assert!(virt.into_usize() == 0x123ABCusize);

Неккорректные адреса

Virt::new() возвращает ошибку ku::error::Error::InvalidArgument при попытке задать некорректный с точки зрения архитектуры x86-64 виртуальный адрес.

let bad_address = 1 << 63;
assert!(Virt::new(bad_address) == Err(InvalidArgument));

Преобразования между Virt и VirtAddr

Virt поддерживает преобразования в/из x86_64::addr::VirtAddr. Это нужно для использования структур из x86_64, например x86_64::structures::tss::TaskStateSegment и x86_64::structures::idt::Entry::set_handler_addr().

fn f(virt_addr: VirtAddr) -> Virt {
    virt_addr.into()
}

let virt = Virt::new(0x123ABC)?;
let virt_addr: VirtAddr = virt.into();
let virt2: Virt = virt_addr.into();
assert!(virt == virt2);
assert!(virt == f(virt.into()));
