Expand description

Тег, позволяющий различать виртуальные адреса ku::memory::Virt и физические ku::memory::Phys, а также (виртуальные) страницы ku::memory::Page и (физические) фреймы ku::memory::Frame.

Required Associated Constants

source

const HEX_PREFIX: &'static str

Заменяет префикс 0x при печати на:

  • 0p (physical) для физических адресов;
  • 0v (virtual) для виртуальных адресов.
source

const ADDR_NAME: &'static str

Задаёт имя типа при печати:

  • "Phys" для физических адресов;
  • "Virt" для виртуальных адресов.
source

const FRAGE_NAME: &'static str

Задаёт имя типа при печати:

  • "Frame" для физических фреймов;
  • "Page" для виртуальных страниц.

Required Methods

source

fn new(addr: usize) -> Result<Addr<Self>, Error>

Создаёт Addr — Phys или Virt — по его битовому представлению addr.

Возвращает ошибку Error::InvalidArgument если битовое представление addr не является корректным для адреса целевого типа.

source

fn is_same_area(x: Addr<Self>, y: Addr<Self>) -> bool

Возвращает true если:

  • Адреса x и y виртуальные и находятся в одной и той же половине виртуального адресного пространства.
  • Адреса x и y физические.
