Module kernel::memory::mmu

Дополнительные функции для работы с Memory Management Unit, доступные только ядру.

Structs

PageTableEntry

Запись таблицы страниц.

PageTableFlags

Флаги в записи таблицы страниц. Аналогичны x86_64::structures::paging::page_table::PageTableFlags.

Constants

FULL_ACCESS

Шаблон флагов для страниц, не ограничивающий доступ.

KERNEL_READ

Флаги для страниц, доступных ядру только на чтение.

KERNEL_RW

Флаги для страниц, доступных ядру на чтение и запись.

PAGE_OFFSET_BITS

Количество бит в смещении внутри виртуальной страницы или физического фрейма. Равно двоичному логарифму их размера.

PAGE_TABLE_ENTRY_COUNT

Количество записей в одном узле таблицы страниц.

PAGE_TABLE_INDEX_BITS

Количество бит в номере записи внутри одного узла таблицы страниц.

PAGE_TABLE_INDEX_MASK

Маска для вырезания из адреса индекса в каком-нибудь из узлов таблицы страниц.

PAGE_TABLE_LEAF_LEVEL

Номер листьевого уровня таблицы страниц.

PAGE_TABLE_ROOT_LEVEL

Номер корневого уровня таблицы страниц.

USER

Шаблон флагов для страниц, доступных коду пользователя.

USER_READ

Флаги для страниц, доступных коду пользователя только на чтение.

USER_RW

Флаги для страниц, доступных коду пользователя на чтение и запись.

Functions

flush⚠

Сбрасывает кеш страничного преобразования процессора (Translation Lookaside Buffer, TLB) для заданной страницы page.

read_cr3

Читает из регистра CR3 физический адрес корневого узла текущей таблицы страниц.

write_cr3⚠

Записывает в регистра CR3 физический адрес корневого узла таблицы страниц. При этом процессор переключается в виртуальное адресное пространство, задаваемое этой таблицей страниц.

Type Definitions

PageTable

Узел таблицы страниц. Аналогичен x86_64::structures::paging::page_table::PageTable.
